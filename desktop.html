Akses Melalui Ponsel Anda

Untuk pengalaman terbaik, silakan buka Velora melalui browser di ponsel Anda. Aplikasi ini dirancang khusus untuk memberikan kemudahan akses di mana saja dan kapan saja.

QR Code

Scan dengan ponsel

Scan QR code ini dengan kamera ponsel Anda untuk langsung mengakses Velora

Cara Mengakses:

1

Buka Browser di Ponsel

Gunakan Chrome, Safari, atau browser lainnya di ponsel Anda

2

Kunjungi Website

Ketik alamat website ini atau scan QR code

3

Install sebagai App

Tambahkan ke layar utama untuk akses cepat seperti aplikasi native

Tips:

Velora adalah Progressive Web App (PWA) yang dapat diinstall seperti aplikasi mobile biasa tanpa perlu download dari app store.
